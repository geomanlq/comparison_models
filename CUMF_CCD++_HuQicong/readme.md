# CUMF_CCD++

This repository is an implementation of paper "Parallel CCD++ on GPU for Matrix Factorization". (*GPGPU-10: Proceedings of the General Purpose GPUs*). [[download](https://dl.acm.org/doi/abs/10.1145/3038228.3038240)]

## Prerequisites:

1. CUDA 7.0+
2. C++
